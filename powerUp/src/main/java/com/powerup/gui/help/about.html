Development

This game was developed in a month and a half for a software course at the Computer Science department of the University of Helsinki. It was written in Java, using the Swing library for its graphical interface.

The idea for this game came from an 1993 Mac OS game called Power Players, developed by Greg Francis, which in turn was based on the 1962 board game Acquire, designed by Sid Sackson. No copyright is claimed and no infringement is intended.
